# Energy Consumption EDA & Baseline Forecast

**PowerFlow Energy Project**

This is my first major project as part of my journey to become a Junior AI Engineer. I'm analyzing electricity consumption, generation, prices and weather data from Spain.

### Project Goal
To understand the patterns in energy consumption and build a simple baseline forecasting model using Prophet.

### What I've done so far

- Set up the project structure
- Loaded and explored the energy dataset (`energy_dataset.csv`)
- Converted time column to proper datetime format
- Performed initial Exploratory Data Analysis (EDA)
- Created first visualizations (overall consumption + hourly patterns)

### Git Workflow

I am following a clean Git workflow for this project:

- `main` — stable version of the project
- Feature branches for new functionality (`feature/eda`, `feature/prophet-model`, `feature/streamlit-dashboard`)
